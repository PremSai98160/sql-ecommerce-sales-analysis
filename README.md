# E-Commerce Sales Intelligence

## Business Problem
Analyzing 100K+ real e-commerce orders from Olist, 
a Brazilian marketplace, to uncover revenue trends, 
top performing products, customer segments, and 
delivery performance issues that impact business growth.

## Tools Used
- PostgreSQL
- SQL (CTEs, Window Functions, JOINs, Aggregations)

## Key Business Questions
1. Which product categories generate the most revenue?
2. What is the monthly revenue trend over 2 years?
3. Which sellers have the highest order cancellation rate?
4. What is the average delivery time by region?
5. Which customers have made repeat purchases?
6. What is the revenue contribution of the top 10% of customers?
7. Which months have the highest late delivery rate?
8. What is the rolling 3-month average revenue per category?
9. Which city has the most orders but lowest review score?
10. RFM customer segmentation using SQL only
